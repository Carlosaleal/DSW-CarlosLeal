La Interfaz de Usuario

La Interfaz de Usuario, también llamada UI (User Interface) es el conjunto de controles y canales sensoriales mediante los cuales el usuario puede comunicarse con una máquina. Por ejemplo, en una computadora, la pantalla, el teclado y los altavoces son parte de la interfaz del usuario, pues nos permite que entre o salga la información del equipo. Una buena interfaz de usuario se caracteriza por tener un alto grado de usabilidad, ser intuitiva y amigable.

Tipos de interfaz:

  • Interfaz Grafica de Usuario (GUI: Graphic User Interface): es el contenido grafico mediante el cual se visualiza la información en una pantalla. Casi todos los programas tienen alguna clase de interfaz visual, que sirve al mismo tiempo para mostrar información al usuario y como un mapa de navegación entre los comandos del programa.

    • Fotografia 1: Interfaz Grafica de Usuario

  • Interfaz de usuario de sitios Web: se refiere a los gráficos, información y herramientas que aparecen en la pantalla al cargar o abrir una pagina web. Estos deben ser muy intuitivos y fáciles de usar, pues sería muy poco práctico que los usuarios o internautas tuvieran que invertir tiempo en aprender a manejar los comandos que se le presentan en el sitio web.

    • Fotografia 2: Interfaz de usuario de Sitios Web

Otros tipos de interfaz de usuario:

  • Interfaz de voz (VUI): Se trata de programas capaces de identificar e interpretar el habla. El ejemplo más claro que tenemos es el reciente auge de las inteligencias artificiales, como Siri, que se controlan por medio de la voz.

  • Interfaz de texto: Se utiliza principalmente en el ámbito de la programación de sistemas operativos y es la evolución de la interfaz de línea de comando primitiva que usaban los primeros programas de computadora.

  • Interfaz natural: Se le llama así al tipo de interfaz que identifica e interpreta acciones naturales del ser humano, como movimientos y expresiones faciales. Un ejemplo de ello son los Videosjuegos con interfaz kinética.

  • Interfaz cerebro-ordenador: Es el tipo de interfaz más innovadora que existe hasta el momento, y aunque aún no cuenta con muchas aplicaciones cotidianas, se está utilizando para controlar prótesis biónicas y dar instrucciones sencillas a un software por medio de las ondas cerebrales.
