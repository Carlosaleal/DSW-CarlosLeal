Bienvenido al sitio web

Se crea este sitio web para adentrarnos en el estudio de las Interfaces de usuario y para estudiar la Interaccion Humano-Ordenador o Humano-Computadora; todo esto como ejercicio de la fase 3: Diseño y construccion de sitios web del curso Diseño de Sitios Web de la UNAD.

Proposito del sitio

Este sitio tiene como propósito comprender los conceptos de Interfaz de usuario y sus tipos, así como el concepto de Interacción Humano-Ordenador o Interacción Humano-Computador; esto como un componente esencial para el aprendizaje del curso Diseño de Sitios Web.

¿Qué es la interfaz del usuario o UI?

De forma sencilla, podemos definirla como el medio a través del cual el usuario interactúa con un dispositivo tecnológico. Esto abarca todos los puntos de contacto entre la persona y el equipo. Para comprenderlo mejor, debes estar consciente de que cada vez que abres el correo electrónico, realizas una transferencia electrónica, envías un emoticono o cualquier otra acción similar estas comunicándote con un software y sistema operativo. Básicamente, transmites o señalas lo que deseas hacer y este responde. ¡Esto lo permite la interfaz del usuario!

Interaccion hombre-Computador

La Interacción Humano-Computador (IHC) es el estudio del diálogo entre el hombre y la computadora, la naturaleza de estos dos interlocutores hace de la IHM un campo multidisciplinaria tales como la informática, la ergonomía y la ingeniería de software entre otros.
